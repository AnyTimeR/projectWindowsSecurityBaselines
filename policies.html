Windows Security Baselines > Policies

Sua ferramenta para segurança e conformidade no Windows

Como usar esta ferramenta:

  1. Obtenha o arquivo de baseline: Baixe as baselines de segurança do Windows no site oficial da Microsoft: Microsoft Security Compliance Toolkit .
  2. Encontre o arquivo .htm: Após baixar e extrair o pacote de baselines, navegue até a pasta: Downloads\Windows 11 Security Baseline\Windows11-Security-Baseline-FINAL\GP Reports.
  3. Importe o arquivo: Clique no botão abaixo para carregar o arquivo .htm da pasta mencionada.
  4. Visualize as políticas: Após a importação, as políticas de nível de Computador e Configurações do Windows serão exibidas em cards.

Observação: Somente políticas de nível de computador e Configurações do Windows (Windows Settings) são filtradas nesta ferramenta. As políticas referem-se ao recurso Group Policy Management.
